Welcome to NoisyTPM. It is a proof-of-concept demonstration of a server-client communication using Noise Framework and TPM attestation to provide a secure communication channel.

This project was developed by Tomáš Babulík, Jiří Rösch and Matěj Groman as a part of course PV204. Source code and more information can be found on Github.
